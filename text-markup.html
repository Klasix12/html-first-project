С помощью атрибута start можем поменять стартовый элемент у упорядоченного списка
С помощью атрибута reversed можем перевернуть список
С помощью атрибута type можем поменять типы маркеров:

  • 1 десятичное число
  • a буквы латинского алфавита в нижнем регистре
  • A буквы латинского алфавита в верхнем регистре
  • i римские цифры в нижнем регистре
  • I римские цифры в верхнем регистре

Также теги <ol> и <ul> можно складывать друг в друга для получения вложенных списков.

    хихи хехе                       хихи хехе

&lt; - знак меньше
&gt; - знак больше

<code> - для обозначения фрагментов кода. Мы можем использовать его с тегом <pre> чтобы отобразить на странице куски кода
<q> - для цитат. Атрибутом cite можно указать источник
<cite> (не атрибут) - расширенная версия атрибута cite. В нем, помимо источника, указывается название произведения, а также имя автора или организации, чей текст цитируется.

<br> - перенос строки
<sub> и <sup> - отображение текста в виде верхнего и нижнего индексов. Используется, в основном, для формул

20м2 H2O X3+X2=1

<time> - используется для разметки даты и времени. Для нас дата описывается внутри тега, а для компьютера - в атрибуте datetime

09:54 утра 18 ноября 2015 в прошлую субботу вчера

<em> - определяет текст, на который сделан особый акцент, меняющий смысл всего текста
<i> - применяется для определения текста, который отличается от окружающего текста, но не является более важным. Как пример, название, термины, иностранные слова, мысли героя
<strong> - указывает на важность выделенного текста
<b> - используется для привлечения внимания, но без придания тексту особой важности
<del> - выделяет текст, который был удален в новой версии документа. В браузере этот текст перечеркивается
<ins> - выделяет текст, который был добавлен в новой версии документа. В браузере этот текст подчеркивается
<del> и <ins> имеют особый атрибут datetime, в котором можно указать, когда были внесены изменения <div> - обычно используется для группировки крупных элементов, например, нескольких абзацев или в качестве контейнера для создания сеток страницы
<span> - для выделения мелких текстовых элементов: частей слов, отдельных слов или фраз, состоящих из нескольких слов
